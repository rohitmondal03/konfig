# Getting Started

Welcome to Konfig! This guide will help you set up your first project, create configurations, and access them from your code.

## 1. Create an Account

First, sign up for a free account at [Konfig Cloud](https://#). 
Once you have created an account, you'll be directed to your dashboard where you can see all your projects.

## 2. Create a Project

A **Project** in Konfig acts as a container for your environments and variables.
1. Click on **New Project**.
2. Give your project a name, for example `my-awesome-app`.
3. Once created, you will be taken to the project overview.

## 3. Add Variables

You can now start adding your environment variables. Variables are grouped by environments (e.g., Development, Staging, Production).

1. Go to the **Variables** tab.
2. Click **Add Variable**.
3. Set the key (e.g., `DATABASE_URL`) and its value.
4. Hit **Save**.

## 4. Generate an API Key

To use these variables in your application, you need an API key.

1. Go to the **API Keys** section within your project.
2. Click **Create API Key**.
3. Select the environment this key should access (e.g., Development).
4. Copy the generated key. **Keep this key safe!**

## 5. Install the SDK

Now that your configurations are set up, you can integrate Konfig into your application. 

Proceed to the [Installation](/installation) guide to install the SDK and start fetching your variables.
